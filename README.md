# Monopoly Deal Mania 🃏

Single-player Monopoly Deal card game with AI opponents.

## Features
- Full 110-card original Monopoly Deal rules
- 1-4 AI opponents with 4 difficulty levels
- Just Say No reaction system
- Wild card rearrangement
- Persistent stats via localStorage
- Mobile-first PWA (Add to Home Screen)

## Deploy on Render
1. Push to GitHub
2. Connect repo on Render as a Web Service
3. Build command: `npm install`
4. Start command: `node server.js`

## Local Development
```bash
npm install
node server.js
# Open http://localhost:3000
```
